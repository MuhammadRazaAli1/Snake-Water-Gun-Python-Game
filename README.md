Gun Water Snake Game
A simple and fun command-line game where the player competes with the computer in a **Gun–Water–Snake** match.  
It’s inspired by the classic *Rock–Paper–Scissors* game and built using basic Python logic.
  
Technologies Used
* Python
* "random" module for computer moves
* Basic conditionals and loops

How to Play
* Run the `mein.py` file.  
* Choose between Gun (g), Water (w), Snake (s).  
* The computer will make a random choice.  
* Results are displayed showing who wins each round.
